# ArquivoIA - Etapa 1: Estrutura Inicial

Olá!

Parabéns, aqui está o código-fonte da primeira etapa do seu site **ArquivoIA**.

Como você não é programador, preparei instruções bem simples. Não se preocupe com termos técnicos, apenas siga os passos. Este projeto foi construído sem a necessidade de ferramentas complexas de instalação.

---

### **Como Visualizar o Site no seu Computador**

Para ver como o site está ficando, você não precisa instalar nada.

1.  **Baixe os Arquivos:** Faça o download de todos os arquivos que eu gerei para uma única pasta no seu computador.
2.  **Abra o Arquivo Principal:** Encontre o arquivo `index.html` e dê um duplo clique nele. Ele vai abrir diretamente no seu navegador de internet (Chrome, Firefox, etc.) e você poderá ver o site funcionando.

---

### **Próximo Passo: Colocar seu Site no GitHub**

Agora, vamos colocar o código na "casa" que preparamos para ele no GitHub. Isso permitirá que o site seja publicado na internet mais tarde.

Siga estes passos com atenção.

**1. Acesse seu Repositório no GitHub**

*   Abra o site do GitHub: [https://github.com](https://github.com)
*   Faça login na sua conta.
*   No canto superior esquerdo, você deve ver seu nome de usuário. Clique nele e vá para **"Your repositories"**.
*   Clique no repositório que criamos anteriormente, chamado **`arquivo-ia`**.

**2. Faça o Upload dos Arquivos**

*   Dentro da página do seu repositório `arquivo-ia`, você verá uma área que diz algo como *"Get started by creating a new file or uploading an existing file"*.
*   Clique no link que diz **"uploading an existing file"**.



**3. Arraste e Solte os Arquivos e Pastas**

*   Uma nova página de upload será aberta.
*   Abra a pasta no seu computador onde você salvou todos os arquivos do site.
*   **Selecione TODOS os arquivos e pastas** (`index.html`, `style.css`, `README.md`, e as pastas `scripts`, `pages`, `components`, `data`).
*   **Arraste todos eles de uma vez** para a área indicada na página do GitHub que diz "Drag files here to add them to your repository".
*   Aguarde alguns segundos até que o GitHub processe e liste todos os seus arquivos.

**4. Confirme o Upload**

*   Depois que os arquivos forem carregados, role a página para baixo. Você verá uma seção chamada **"Commit changes"**.
*   No primeiro campo, você pode digitar uma breve descrição. Escreva: `Criação da estrutura inicial do site`.
*   Deixe as outras opções como estão e clique no botão verde **"Commit changes"**.

**Pronto!** Você acaba de enviar a primeira versão do seu site para o seu repositório no GitHub. Este é um passo fundamental e você o completou com sucesso.
